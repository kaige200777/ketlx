# 更新日志

## [v1.1.0] - 2025-12-12

### 新增功能
- ✨ **AI批改功能** - 支持使用AI自动批改简答题
  - 支持 OpenAI、DeepSeek 等多种 AI 服务
  - 可配置 API 密钥和模型
  - 自动评分和生成评语
  - 超时处理和重试机制

- ✨ **题库导出功能** - 支持将题库导出为 Excel 文件
  - 支持所有题型导出
  - 自动生成文件名（题库名_题型_时间）
  - 保留题目格式和解析
  - 可重新导入系统

- ✨ **启动时AI配置检测** - 服务启动时自动检测 AI 配置状态
  - 检查配置项完整性
  - 测试 API 连接有效性
  - 识别常见错误（无效密钥、超时等）
  - 友好的错误提示

### 改进优化
- 🎨 **优化教师控制面板布局**
  - 调整简答题批改和总分位置
  - 简答题批改与学生自选放在同一行
  - 移除冗余的配置状态信息

- 🔧 **改进错误处理**
  - AI 批改超时处理
  - API 请求重试机制
  - 更详细的错误日志

- 📝 **完善文档**
  - 新增 AI 批改功能使用指南
  - 新增启动 AI 配置检测说明
  - 新增题库导出功能说明
  - 新增生产环境部署指南
  - 新增 GitHub 推送指南
